# Proyecto final – Ramo visualización de datos ambientales

Este repositorio contiene el proyecto final del curso de Visualización de datos ambientales.\
Para hacerlo más divertido voy a explorar datos de hechizos y monstruos de D&D 5ta edición.

# Audiencia objetivo

Mi audiencia serán tanto jugadores de D&D interesados en empezar a utilizar código simple para analizar el universo de DnD, así como principiantes en codear interesados en aprender de una manera llamativa y con datos fuera de lo común.

# Datos

-   "spells": base de datos con información de hechizos (nivel, escuela de magia, clases que lo usan, etc.).
-   "monsters": base de datos con información de monstruos (tipo, alineamiento, puntos de golpe, resistencias, etc.).

En este proyecto se usarán estos datos para hacer preguntas exploratorias y buscar patrones. Ambos datasets fueron sacados de "Data Science Learning Community", y ambos fueron curados por Jon Harmon.
